SandwichMaker - This class invokes the main method of the project. It will keep track of the amont
		of sanwiches the chefs have made and eaten and will stop them when they reach 20.

Chef - 	This class mimics a chef who posesses an unlimited number of one ingredient. Will wait
	for the Agent class to provide them the ingredients necessary to make their sandwich.

Agent - This class mimics a chef who posesses an unlimited number of multiple ingredients.
	Will choose two of the ingredients and will send them to the chef that does not have
	said ingredients.

Setup: Run the main method from the SandwichMaker class.
